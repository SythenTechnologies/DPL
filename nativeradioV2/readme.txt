Installation:

- extrakt the download zip archiv in a folder
- upload all files to your webserver
- use the embeded code from example.html and customized the parameters

variables: swfcolor, swfpause, swfstreamurl
swfstream - direct stream url (no support for container formate: pls, asx, etc.)
swfcolor - the RGB hex value for colorized radio player
swfpause - direct start the radio stream after loading the player (=0) or pause (=1)

optionales variables: swfbuffer, swfradiochannel, swfinformations
swfbuffer - browser buffer for reconnect, default value is 50 (MB)
swfradiochannel - alternative text for radio channel name, if server cross-scripting is not possible 
swfinformations - alternative text for scound text field, if server cross-scripting is not possible
swfintervall - timer intervall for reading the song informations in sec.
